## Module: Authentication

### What This Module Does

WDI V2 uses magic link as the primary authentication method for all roles.
Password is an optional secondary (opt-in only — no passwords are migrated
from V1). There are no exceptions to magic link for any role.

On-screen link display is the deliverability fallback when email is blocked
by a receiving mail server (copy-to-clipboard for manual delivery).

### Locked Decisions (authoritative — do not deviate)

| ID | Decision |
|----|----------|
| D-ARCH-016a | Magic link = primary auth; password = optional secondary (opt-in, no V1 migration) |
| D-ARCH-047 | Magic link expiration: 30 minutes, single-use enforcement |
| D-ARCH-057 | Magic link resend on-screen: copy-to-clipboard for manual delivery |
| D-ARCH-058 | No password fallback if no password set — direct to magic link |
| D-ARCH-017 | Role-based session timeouts: AP/CF = 48 hours; TA/FA/CA = 12 hours |
| D-ARCH-018 | Keep-me-logged-in flag: extends session to role-based timeout limit |
| D-ARCH-019 | Cookie-based sessions: httpOnly + secure + sameSite=strict. NOT JWT. |
| D-AUTH-001 | Unified auth token table: magic link + password reset share one table (type field) |
| D-AUTH-002 | Password change requires no current password (authenticated session = sufficient) |
| D-AUTH-003 | Password set/reset: 3 entry points (forgot, account screen, TA/FA-initiated) |
| D-AUTH-004 | Password login gate: blocks password login if no password set, directs to magic link |
| D-AUTH-005 | FA user management scope: FA can reset password, activate/deactivate CA/CF users |

### Roles That Use Auth

| Role | Code | Session Timeout |
|------|------|----------------|
| Tenant Admin | TA | 12 hours |
| Fund Administrator | FA | 12 hours |
| Campus Advocate | CA | 12 hours |
| Campus Focal | CF | 48 hours |
| Applicant (external) | AP | 48 hours |

### User Entity (auth-relevant fields only)

```
users
  id              UUID PK
  email           VARCHAR UNIQUE NOT NULL
  role            ENUM(TA, FA, CA, CF, AP)
  modules[]       ARRAY — module access control
  has_password    BOOLEAN DEFAULT false
  is_active       BOOLEAN DEFAULT true
  keep_logged_in  BOOLEAN DEFAULT false
  last_login_at   TIMESTAMP
  created_at      TIMESTAMP
  updated_at      TIMESTAMP
```

### Auth Token Entity

```
auth_tokens
  id          UUID PK
  user_id     UUID FK → users.id
  token       VARCHAR UNIQUE NOT NULL (hashed)
  type        ENUM(magic_link, password_reset)
  expires_at  TIMESTAMP (30 min from creation)
  used_at     TIMESTAMP NULL (NULL = unused)
  created_at  TIMESTAMP
```
